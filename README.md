#Installation

1. Install Adonis<br>
`npm i -g @adonisjs/cli`

2. Clone project<br>
`git clone https://github.com/AvetisDN/adonisjs.git`

3. Generate .ENV file<br>
`adonis key:generate`

#############################################################

4. Configure DB connection in .ENV file

5. Run migrations<br>
`adonis migration:run`

6. Seeding DB data<br>
`adonis seed`
